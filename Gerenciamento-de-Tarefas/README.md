# Gerenciamento de Tarefas
Um projeto efetuado na linguagem Python para gerenciar tarefas desejadas. O projeto é capaz de:
- Adicionar tarefas
- Ver tarefas
- Atualizar tarefas
- Completar tarefas
- Deletar tarefas

Inspiração: Projeto de finalização do Módulo 1 - Formação Python (Rocketseat)
